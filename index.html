1. Logic and Proof: Necessary & Sufficient Conditions

This area is crucial for understanding causality and logical constraints in economic theory.

ConceptNotationInterpretationCondition Type
Implication$$ A \rightarrow B $$"If $A$, then $B$." ($A$ leads to $B$).$A$ is the sufficient condition for $B$. $B$ is the necessary condition for $A$.
Equivalence$$ A \leftrightarrow B $$"$A$ if and only if $B$ (iff $B$)."$A$ is necessary and sufficient for $B$. (Assignment Q1, Week 0).
Converse$$ B \rightarrow A $$"If $B$, then $A$." (The reverse).Not logically equivalent to $A \rightarrow B$.
Contrapositive$$ \neg B \rightarrow \neg A $$"If not $B$, then not $A$."Logically equivalent to $A \rightarrow B$.

Assignment Application (Q3, Week 1):

  • Implication: $$(x-1)(x-2)(x-3)=0 \rightarrow x=1$$ This is FALSE. If $x=2$, the first part is true, but the second part ($x=1$) is false.
  • Converse: $$x=1 \rightarrow (x-1)(x-2)(x-3)=0$$ This is TRUE. If $x=1$, the expression equals $(0)(-1)(-2) = 0$.

2. Functions, Real Numbers, and Inequalities

This covers the valid domain of functions and solving fundamental inequalities.

A. Inequalities and Absolute Value

ConceptFormulaRule / Solution Method
Absolute Value$$|x-a| < k$$This expands to: $$-k < x-a < k$$

Assignment Application (Q1, Week 1):

Problem: Solve $$|x+1| < 5$$

Solution: $$-5 < x+1 < 5$$ Subtract 1 from all parts: $$-5 - 1 < x < 5 - 1 \quad \rightarrow \quad \mathbf{-6 < x < 4}$$

B. Domain and Undefined Points

ConceptFormulaCondition for Real DomainUndefined Condition
Square Root Function$$ f(x) = \sqrt{g(x)} $$Term under root must be non-negative: $$ \mathbf{g(x) \ge 0} $$N/A
Rational Function$$ h(x) = \frac{f(x)}{g(x)} $$N/AUndefined when denominator is zero: $$ \mathbf{g(x) = 0} $$

Assignment Application:

  • Domain (Q1, Week 3): For $g(x) = \sqrt{x-1}$, require $x-1 \ge 0$, thus $\mathbf{x \ge 1}$.
  • Undefined Points (Q8, Week 1): For $\frac{x^2+x}{x(x+1)(x-4)}$, undefined where $x(x+1)(x-4) = 0$. Occurs at $\mathbf{x=0, x=-1, \text{ and } x=4}$.

3. Set Theory and Budget Constraints

Used for modeling consumption bundles and aggregating groups.

A. Set Operations and Counting Rules

ConceptFormulaMeaningAssignment Application
Union Size$$ n(A \cup B) = n(A) + n(B) - n(A \cap B) $$Elements in $A$ or $B$ (or both).Calculating total participants in sports (Week 2).
Difference$$ A \setminus B $$Elements in $A$ but not in $B$.Q1 (Week 2): $G \setminus H$ (Graduates not in Hostel).
Symmetric Difference$$ n(A \Delta B) = n(A \cup B) - n(A \cap B) $$Elements in $A$ or $B$, but not both.Calculating participants in only one sport (Week 2).
Total Subsets$$ \mathbf{2^n} $$Number of subsets for a set with $n$ elements.Q8 (Week 2): Set with 4 elements has $2^4 = \mathbf{16}$ subsets.

B. Budget Constraint (Microeconomics)

The boundary of feasible consumption for goods $x, y$ with prices $P_x, P_y$ and income $I$.

  1. Budget Constraint (Inequality): Expenditure $\le$ Income. $$ \mathbf{P_x x + P_y y \le I} $$
  2. Budget Set (Set Notation): All affordable bundles. $$ \{(x, y): P_x x + P_y y \le I, x \ge 0, y \ge 0\} $$

Assignment Application (Q6, Week 2):

  • Given: $I=800$, $P_x=20$, $P_y=10$.
  • Constraint: $$ 20x + 10y \le 800 \quad \rightarrow \quad \mathbf{2x + y \le 80} $$

The core of marginal analysis and finding maxima/minima in economics.

5. Macroeconomics and Dynamic Models

Modeling aggregate income and its change over time.

A. Simple Income Determination (Macro Model)

  1. Equilibrium Condition: Aggregate Supply = Aggregate Demand. $$ Y = C + I $$
  2. Consumption Function: $$ C = a + bY $$ ($b$ = Marginal Propensity to Consume, $0 < b < 1$)
  3. Equilibrium Income: (Assuming $I = \bar{I}$) $$ Y = \frac{a}{1-b} + \frac{\bar{I}}{1-b} $$

B. Time Path of Variables (Dynamics)

Solution for a first-order linear difference equation.

ConceptFormulaStability ConditionType of Convergence (if stable)
Time Path$$ \mathbf{P_t = (P_0 - P^*)b^t + P^*} $$$$ \mathbf{|b| < 1} $$Monotone: If $b > 0$.
Oscillating: If $-1 < b < 0$.
Equilibrium Price$\mathbf{P^*}$$P^*$ is the Intertemporal Equilibrium Price (Particular Integral).
