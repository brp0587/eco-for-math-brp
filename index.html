2. Functions, Real Numbers, and Algebra

This section details the fundamental rules governing real numbers, algebraic operations, and function properties necessary for economic modeling.

A. Function Domain Rules

The domain is the set of valid inputs ($x$) for a function $f(x)$ to return a real number.

  • Square Root: For $f(x) = \sqrt{g(x)}$, the domain requires $\mathbf{g(x) \ge 0}$.
  • Rational Function: For $f(x) = \frac{g(x)}{h(x)}$, the domain requires $\mathbf{h(x) \ne 0}$.
  • Logarithm: For $f(x) = \ln(g(x))$, the domain requires $\mathbf{g(x) > 0}$.

B. Inequalities and Absolute Value

InequalityExpansion / Solution
$$ |x| < k \quad (k>0) $$$$ -k < x < k $$
$$ |x| > k \quad (k>0) $$$$ x < -k \text{ or } x > k $$
Quadratic Formula (for $ax^2+bx+c=0$)$$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$

3. Set Theory and Economic Sets

Understanding sets is essential for defining economic spaces, such as consumption bundles and probability spaces.

A. Core Set Operations

ConceptNotationFormula / Meaning
Union$$ A \cup B $$Elements in $A$ or $B$.
Intersection$$ A \cap B $$Elements in $A$ and $B$.
Difference$$ A \setminus B $$Elements in $A$ but not in $B$.
Total Subsets$$ 2^n $$Number of subsets for a set with $n$ elements.
Size of Union$$ n(A \cup B) = n(A) + n(B) - n(A \cap B) $$The Inclusion-Exclusion Principle.

B. Budget Set Definition (Consumption)

For two goods ($x_1, x_2$) with prices ($P_1, P_2$) and income $I$:

  • Budget Constraint: The limit on expenditure. $$ P_1 x_1 + P_2 x_2 \le I $$
  • Budget Set: The collection of all affordable consumption bundles. $$ \{(x_1, x_2): P_1 x_1 + P_2 x_2 \le I, x_1 \ge 0, x_2 \ge 0\} $$

4. Differentiation, Extrema, and Integration

The mathematical toolkit for marginal analysis, elasticity, and optimization in both micro- and macroeconomics.

A. Core Differentiation Rules

RuleFunction / OperationDerivative
Power Rule$$ y = x^n $$$$ \frac{dy}{dx} = nx^{n-1} $$
Product Rule$$ y = f(x)g(x) $$$$ y' = f'(x)g(x) + f(x)g'(x) $$
Quotient Rule$$ y = \frac{f(x)}{g(x)} $$$$ y' = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2} $$
Chain Rule$$ y = f(g(x)) $$$$ y' = f'(g(x)) \cdot g'(x) $$
Exponential Rule$$ y = e^{ax} $$$$ y' = ae^{ax} $$

B. Optimization (Extrema)

ConceptCondition / RuleNotes
FOC (Necessary)$$ f'(c) = 0 $$First Order Condition: Required for a local max or min at an interior point $c$.
SOC (Sufficient for Max)$$ f''(c) < 0 $$Second Order Condition: Confirms a local **Maximum** at $c$.
SOC (Sufficient for Min)$$ f''(c) > 0 $$Second Order Condition: Confirms a local **Minimum** at $c$.
Marginal Cost vs. Average Cost$$ AC(q^*) = MC(q^*) $$Average Cost is minimized ($AC'$=0) when Marginal Cost equals Average Cost.

C. Integration

Integration is used to find total functions from marginal functions (e.g., Total Cost from Marginal Cost).

  • Integration by Parts: $$ \mathbf{\int u dv = uv - \int v du} $$
  • Fundamental Theorem: If $F'(x) = f(x)$, then $\int_{a}^{b} f(x) dx = F(b) - F(a)$.

5. Macroeconomics and Dynamic Models

This section focuses on the simple Keynesian model and dynamic analysis of economic variables over discrete time periods.

A. Simple Keynesian Model

  • Consumption Function: $C = a + bY$ (where $b = MPC$, $0 < b < 1$).
  • Equilibrium Condition: Aggregate Supply ($Y$) = Aggregate Demand ($C+I$). $$ \mathbf{Y = \frac{a+I}{1-b}} $$
  • Multiplier: The change in equilibrium income ($\Delta Y$) resulting from a change in autonomous spending ($\Delta I$). $$ \mathbf{\frac{1}{1-b}} $$

B. First-Order Linear Difference Equations

General solution for $Y_t = bY_{t-1} + a$.

  • Time Path Solution: $$ \mathbf{Y_t = (Y_0 - Y^*)b^t + Y^*} $$
  • Equilibrium Value (Particular Integral): $$ \mathbf{Y^* = \frac{a}{1-b}} $$
  • Stability Condition: The system converges to equilibrium if the coefficient $b$ satisfies: $$ \mathbf{|b| < 1} $$
  • Type of Convergence: Monotone if $0 < b < 1$; Oscillating if $-1 < b < 0$.

6. Financial Mathematics

Key formulas for calculating future and present values under different compounding scenarios.

A. Compounding and Discounting

ConceptFormulaNotes
Future Value (Discrete)$$ \mathbf{FV = P(1+r)^t} $$$P=$ Principal, $r=$ rate per period, $t=$ number of periods.
Compounding $n$ times/year$$ \mathbf{FV = P\left(1+\frac{r}{n}\right)^{nt}} $$$r=$ annual rate, $n=$ compounding frequency.
Present Value (Discounting)$$ \mathbf{PV = \frac{FV}{(1+r)^t}} $$Used to find today's value of a future amount.

B. Continuous Compounding

  • Limit Definition of $e$: $$ \mathbf{\lim_{n\rightarrow\infty}\left(1+\frac{1}{n}\right)^{n} = e} $$
  • Future Value (Continuous): $$ \mathbf{FV = Pe^{rt}} $$
  • Rule of 70/72: Time to double (in years) $\approx \frac{70}{\text{Growth Rate (\%)}}$.
